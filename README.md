# Prajs-hideout
My personal website, all abt me :D

# Description

This is my personal portfolio website where I showcase my skills, projects, and interests in technology. The website serves as a central place to present my work, including web development projects, robotics builds, and creative ideas.

It is designed to reflect my personality as a tech enthusiast and creator, while also making it easy for others to explore what I have built.

# How to Try It Out

You can try the project here: https://prajs-hideout.web.app/

# Tools Used
HTML - to make the website.
CSS - to give the style.
JavaScript - to configure the Firebase backend and make the notes logic.
ChatGPT - to help me with Firebase and JS.
terminal - to login, initialise and configure Firebase and deploy my projects into Firebase hosting.

# Features
- A personal introduction section that explains who I am and what I do
- A projects section to showcase my work and creations
- A structured layout that makes navigation simple and clear
- Custom-designed components to present content effectively
- A responsive design that works across different screen sizes
